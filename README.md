# tekmonks - node.js exam
Online coding examination to test the applicants knowledge as a node.js developer.

### Install

    npm install


### Start

    npm start
    
    
## Dependencies

 1. [express.js](https://expressjs.com/)
 2. [nodemon](https://www.npmjs.com/package/nodemon)
